## Lokal oppsett

### Krav

- Node v20 (bruk [nvm](https://github.com/nvm-sh/nvm) eller en annen versjonsbehandler)
- pnpm v9: https://pnpm.io/installation

### Installer avhengigheter

```bash
pnpm i
```

### Kjør lokalt

```bash
pnpm run dev
```

### Distribusjon

```bash
pnpm run build
```

Kopier `build`-mappen til hostingen din.
